# Startup

Boot up sandbox
Open CLI and run: 
    git clone https://github.com/bhite1/aws_opensky.git
    cd aws_opensky/cloudformation
    chmod +x PrimaryDeploymentScript.sh
    ./PrimaryDeploymentScript.sh
Wait ~6 minutes
